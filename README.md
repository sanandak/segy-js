# segy-js
=========

A library to read SU (and eventually SEG-Y) files and return a JSON formatted array of traces.

## Installation

  npm install https://github.com/sanandak/segy-js

## Usage

  var readSU = require('segy-js').readSU
  var traces = readSU('filename')
